Гамильтониан задается в папке params, файл hamiltonian_operators.txt
Сначала вводим кол-во экспонент, для построения анзаца
Затем для каждой вводим оператор Паули.
Здесь используется рандомизация theta с подмешиванием шума для каждого нового параметра theta
Использована библиотека scipy, это готовая библиотека, которая реализует отжиг.
